drawString の統一
	内部コードを unicode にした。
	現在は wchar_t * の drawString は unicode.
	フォント側が SJIS の場合、FontTable で変換する必要がある。
	private drawStringUTF16 と名前を変える予定。
	その場合、外部 API として drawString があり = drawStringUTF16
	となる予定。

	後方互換のために char * の drawString を残した。これは
	ShiftJIS 決めうちとした。pufi++ では使わない方向にする。
	使わないがそのまま残す。
	drawStringSJIS あるいは drawStringASCII などと名前を変える予定。
